Differential Description

The Gaussian function as a solution to the heat equation...

$${\frac {\partial u}{\partial t}}={\frac {\partial ^{2}u}{\partial x_{1}^{2}}}+\cdots +{\frac {\partial ^{2}u}{\partial x_{n}^{2}}}$$

Eigenfunction Description

The Gaussian function as the eigenfunction of the Fourier transform...

$$\hat{f}(\xi) =-\int _{-\infty }^{\infty }f(x)e^{-2\pi i x \xi}dx$$

Variational Description

The Gaussian function as a solution to a energy-entropy maximization...

$$H(X)=-\int _{-\infty }^{\infty }f(x)\log f(x)dx$$

The Gaussian function as the limiting distribution of a stochastic process (Central Limit Theorem)...

$${\sqrt {n}}\left({\bar {X}}_{n}-\mu \right)\ \xrightarrow {d} \ {\mathcal {N}}\left(0,\sigma ^{2}\right)$$
